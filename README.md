# タイトル
「その本、メルカリで！」

# 概要
amazonで見ている本が、メルカリで何円なのか分かります。
(Chrome拡張ツール)
![screenshot](screenshot.png)

# 詳細
amazonで見ている本が、メルカリで出品されている場合、その価格を表示します。
1円でも安く、その本が買えるかもしれません！
